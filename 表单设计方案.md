## 阅读说明

### 控件类型图例

| 标记 | 控件 | 说明 |
|------|------|------|
| `INPUT` | 单行文本框 `el-input` | 字符串类短文本 |
| `TEXTAREA` | 多行文本框 `el-input(textarea)` | 长文本，默认 4 行 |
| `NUMBER` | 数字输入框 `el-input-number` | 纯数值 |
| `DATE` | 日期选择器 `el-date-picker(date)` | 单个日期 |
| `DATERANGE` | 日期区间 `el-date-picker(daterange)` | 起止日期对 |
| `RADIO` | 单选按钮组 `el-radio-group` | 二选一或少量选项 |
| `CHECKBOX` | 多选框组 `el-checkbox-group` | 多选选项 |
| `PERCENT` | 百分比只读 `el-input(readonly)` | 系统自动计算 |
| `UPLOAD` | 文件上传 `el-upload` | 附件上传 |
| `TABLE` | 可编辑动态列表 | 可添加/删除行的表格 |

### 必填标记

- ✅ **必填**：提交时强制验证，为空则阻止提交
- ⭕ **条件必填**：当父字段选择"是"或特定值时变为必填
- ☐ **选填**：可留空

### 字段间逻辑说明

- `[条件显示]`：仅当指定条件成立时该字段才可见
- `[自动计算]`：值由其他字段计算得出，只读
- `[附件上传]`：需同时上传对应证明文件
